jsonata-extended

Extended JSONata object with custom functions

This can be a drop in substitue for the standard JSONata

It is NOT published to NPMJS and so install from this repo

Installation

npm install https://github.com/martinholden-skillsoft/jsonata-extended

Example

// const jsonata = require('jsonata');
const jsonata = require('jsonata-extended');
const expr = jsonata(
  '$htmltotext("<p>Leadership has a dark side; a "leadership shadow" that often creates an unknown; lurking fear.</p>")'
);
const result = expr.evaluate();
